# Wrap Text — Support

  Support and discussions for **Wrap Text**, the free Canva app that wraps text around any image silhouette.

  ## Get Help

  - **FAQs**: [wraptext.app/support](https://wraptext.app/support)
  - **Privacy Policy**: [wraptext.app/privacy](https://wraptext.app/privacy)
  - **Terms**: [wraptext.app/terms](https://wraptext.app/terms)

  ## How to Use

  1. Open any design in Canva
  2. Select an image on your canvas (PNG with transparent background works best)
  3. Open Wrap Text from the Apps panel
  4. Paste your text and click **Wrap Text**
  5. Text flows around the image silhouette — fully editable native Canva text
